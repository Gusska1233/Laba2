## Задача 1

### Текст задачи

Получить список из максимальных цифр натуральных чисел, содержащихся в
исходном списке

### Алгоритм решения

Программа начинается с главной функции main. Сначала main вызывает функцию generateList с аргументом 5, которая формирует список чисел, введенных пользователем. Внутри generateList создается список с помощью генератора списка: цикл for i in 1..count выполняет пять итераций, на каждой из которых программа выводит приглашение "Введите ЧИСЛА для формирования списка:", считывает строку из консоли с помощью Console.ReadLine, преобразует её в целое число через int и добавляет значение в список с помощью yield.

Затем main применяет функцию maxDigit к каждому элементу списка list1 с помощью List.map, сохраняя результат в новую значение newList. Функция maxDigit принимает целое число и находит его максимальную цифру. Внутри maxDigit определена рекурсивная функция loop, которая принимает текущее обрабатываемое число и текущий максимальный элемент. Если исходное число положительное, то loop проверяет: если текущее число стало равно нулю, значит все цифры обработаны и функция возвращает накопленный максимум, иначе она извлекает последнюю цифру через остаток от деления на 10, сравнивает её с текущим максимумом, определяет новое значение максимума и рекурсивно вызывает себя с числом, разделенным на 10 (отбрасывая последнюю цифру), и новым максимумом. Если исходное число отрицательное, функция сначала преобразует его в положительное с помощью унарного минуса, а затем применяет ту же логику обработки. Функция loop инициализируется с максимальным элементом равным 0. После применения maxDigit ко всем элементам, main выводит полученный список максимальных цифр на экран.

## Задача 2

### Текст задачи

Список содержит строки. Найти суммарную длину этих строк.

### Алгоритм решения

Программа начинается с главной функции main, которая сначала вызывает функцию generateStringList с аргументом 5. Внутри generateStringList аналогично создается список с помощью генератора: цикл на пять итераций, на каждой выводится приглашение "Введите СЛОВО для формирования списка:", считывается строка из консоли с помощью Console.ReadLine, преобразуется в строковый тип и добавляется в список через yield.

Далее main вычисляет суммарную длину всех строк с помощью List.fold. Функция fold начинает с начального значения 0 и проходит по каждому элементу списка stringList. На каждой итерации к накопленной сумме прибавляется длина текущей строки, получаемая через свойство Length. Аккумулятор sum обновляется, и после обработки всех элементов fold возвращает общую сумму длин. Эта сумма сохраняется в значение lengthSum. В конце main выводит результат на экран.
